# 👋 Intro

> 게임의 깊은 재미와 섬세한 연출에 진심으로 매력을 느끼는 프로그래머 김동건입니다.   
> 게임의 재미를 극대화하는 연출과 시스템 구현에 기여하는 동시에, 문제 해결 중심의 사고를 바탕으로   
> 개발 과정에서 발생하는 다양한 문제들을 적극적으로 해결해 나가며 끊임없이 배우고 성장하는 개발자가 되겠습니다.
